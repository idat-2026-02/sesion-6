¿Qué es un Hipervínculo?

Es el elemento fundamental de la Web (la "H" en HTML: HyperText). Permite conectar una página con otra o con recursos externos.

Propósito

Navegar entre documentos y crear una experiencia conectada.

Interactividad

Transforma un texto o imagen estática en un punto de acción.

Anatomía de la etiqueta <a>

Se compone principalmente del atributo href (Hypertext Reference).

<a href="https://idat.edu.pe" target="_blank">
  Visitar sitio oficial
</a>
  • 🔹 href: URL o ruta del destino del enlace.
  • 🔹 target: Define dónde se abre (_blank para nueva pestaña).
  • 🔹 title: Información adicional al pasar el mouse.

¿Qué es un <img>?

Es la etiqueta utilizada para incrustar imágenes en un documento HTML. A diferencia de otras etiquetas, es un elemento vacío (no tiene etiqueta de cierre).

Visualización

Permite mostrar formatos como JPG, PNG, GIF, SVG y WebP.

Naturaleza

Es un elemento de línea por defecto, pero se comporta como bloque reemplazado.

Anatomía de la etiqueta <img>

Requiere obligatoriamente dos atributos para ser accesible y funcional:

<img src="logo-idat.png" alt="Logotipo de IDAT" width="200">
  • 🔹 src (Source): La ruta o URL de la imagen que queremos mostrar.
  • 🔹 alt (Alternative text): Descripción textual para lectores de pantalla o si la imagen falla.
  • 🔹 width / height: Atributos opcionales para definir dimensiones (en píxeles).

Tipos de Enlaces

Enlaces Externos

Apuntan a dominios fuera de tu sitio. Siempre usa URLs absolutas (con http://).

Enlaces Internos

Navegan entre páginas del mismo proyecto (ej. contacto.html).

Anclas (Bookmarks)

Saltan a una sección específica de la misma página usando el ID del elemento.

Protocolos Especiales

Podemos disparar acciones del sistema directamente desde el navegador:

Correo electrónico:

href="mailto:soporte@idat.pe"

Llamada telefónica:

href="tel:+51999888777"

Mensaje WhatsApp:

href="https://wa.me/número"

Descarga de archivo:

<a href="..." download>

Estructura Semántica de Menú

Para la accesibilidad y SEO, los menús de navegación deben ir dentro de la etiqueta <nav>.

<nav>
  <ul>
    <li><a href="index.html">Inicio</a></li>
    <li><a href="servicios.html">Servicios</a></li>
  </ul>
</nav>
